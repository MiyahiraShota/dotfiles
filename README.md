# dotfiles

## 設置
```
cd ~
git clone git@github.com:MiyahiraShota/dotfiles.git .dotfiles
cd .dotfiles
make
```

## 必要なやつ
zsh

*あると便利*
tmux
fzf

## zshのみ
```
make set_zsh_env && make set_zsh
```
